# Mathematics with Python
Charts, mathematical methods, and the interface with Python

Includes:
1. An example of processing an arithmetic function and displaying its value
2. An example of computing a function value using a power series
3. Plotting graphs of single-variable functions
4. Plotting graphs of two-variable functions
5. Finding the roots of a single-variable function
6. An algorithm for determining the point of the global maximum of a single-variable function
7. A Monte Carlo method for estimating the area of a region
8. Estimating functional dependencies using the least squares method
9. Creating a simple interface for performing computations
